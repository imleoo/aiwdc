Getting Started with AiWDC: Complete Setup Guide

Learn how to set up your first AI content pipeline in under 10 minutes. This comprehensive guide covers everything from account creation to your first automated workflow.

Alex Chen Technical Writer

  • Overview
  • Account Setup
  • Creating Your First Workflow

  • Testing Your Pipeline

Overview

Welcome to AiWDC! This guide will walk you through setting up your first AI-powered content pipeline. By the end of this tutorial, you'll have a fully automated system that can:

  • Connect to your favorite data sources
  • Process content using advanced AI models
  • Automatically publish to multiple platforms
  • Run continuously without manual intervention

üí° What You'll Need

  • An AiWDC account (sign up for free trial)
  • Access to at least one data source (RSS feed, website, etc.)
  • A publishing destination (WordPress, social media, etc.)
  • About 10 minutes of your time

Step 1: Account Setup

First, let's get your AiWDC account set up and ready to go.

Creating Your Account

  1. Visit aiwdc.com and click "Try AiWDC Cloud"
  2. Choose your preferred sign-up method (email, Google, or GitHub)
  3. Verify your email address if required
  4. Complete the onboarding questionnaire to personalize your experience

Quick Start with CLI (Optional)

# Install AiWDC CLI
npm install -g @aiwdc/cli

# Login to your account
aiwdc login

# Initialize a new project
aiwdc init my-first-pipeline

Step 2: Creating Your First Workflow

Now let's create your first automated content workflow using the visual workflow builder.

Using the Workflow Builder

  1. Navigate to the Dashboard and click "Create New Workflow"
  2. Choose a template or start from scratch
  3. Give your workflow a descriptive name (e.g., "RSS to Blog Automation")
  4. Set your workflow schedule (hourly, daily, or custom)

‚ö†Ô∏è Important Note

Start with a simple workflow for your first setup. You can always add complexity later as you become more familiar with the platform.

Step 3: Connecting Data Sources

Data sources are where your content originates. AiWDC supports multiple types of data sources.

Popular Data Source Options

  • RSS Feeds - Perfect for news sites, blogs, and content aggregation
  • Web Scraping - Extract content from any website
  • Database - Connect to MySQL, PostgreSQL, or MongoDB
  • APIs - Integrate with REST APIs and webhooks

Setting Up an RSS Feed (Example)

  1. In your workflow, click "Add Data Source"
  2. Select "RSS Feed" from the connector list
  3. Enter the RSS feed URL (e.g., https://example.com/feed.xml)
  4. Configure filtering options if needed
  5. Test the connection to ensure it's working

Step 4: AI Model Configuration

This is where the magic happens! Configure your AI model to transform raw data into polished content.

Choosing Your AI Model

Model Best For Speed Quality
GPT-4 High-quality content, complex tasks Medium Excellent
Claude Long-form content, research Medium Excellent
Qwen Multilingual, cost-effective Fast Good

Configuring Content Generation

  1. Select your preferred AI model
  2. Define your content template and style
  3. Set content length and format preferences
  4. Configure tone and voice settings
  5. Add any custom instructions or constraints

Step 5: Setting Up Publishing

Configure where your generated content will be published automatically.

WordPress Integration Example

  1. Click "Add Publishing Destination"
  2. Select "WordPress" from the list
  3. Enter your WordPress site URL and credentials
  4. Configure post settings (categories, tags, status)
  5. Set up scheduling options

‚úÖ Pro Tip

Start with draft mode for your first few runs to review content before it goes live. You can switch to auto-publish once you're confident in your setup.

Step 6: Testing Your Pipeline

Before going live, it's crucial to test your workflow to ensure everything works as expected.

Running a Test

  1. Click "Test Workflow" in the workflow builder
  2. Monitor the execution in real-time
  3. Review the generated content
  4. Check that publishing worked correctly
  5. Make adjustments if needed

Common Issues and Solutions

  • Connection errors - Double-check your credentials and URLs
  • Poor content quality - Adjust your AI prompts and templates
  • Publishing failures - Verify destination platform permissions
